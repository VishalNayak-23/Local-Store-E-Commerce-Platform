# 🛒 Local Store E-Commerce Platform

A fully functional e-commerce website built with pure HTML, CSS, and JavaScript — no frameworks, no backend. All data including cart, orders, and user accounts are stored in the browser's localStorage.

## Tech Stack

- **Frontend:** HTML5, CSS3, JavaScript (Vanilla JS)
- **Storage:** Browser localStorage (no backend, no database)
- **Tools:** VS Code, Live Server

## Features

- 🛍️ Product listing with search, filter, and sort
- 📦 Product details page
- 🛒 Shopping cart with quantity update and remove
- 💳 Checkout form with validation
- 📋 Order history
- 👤 User register and login (localStorage-based)
- 📱 Fully responsive for mobile and desktop

## How to Run

1. Clone the repository
2. Open the folder in VS Code
3. Install the **Live Server** extension
4. Right-click `index.html` → **Open with Live Server**
